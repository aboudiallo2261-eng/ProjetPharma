SIGNATURES "IA" DÉTECTÉES ET ÉRADIQUÉES
----------------------------------------
1. Titres de page hardcodés : `-fx-text-fill: #2C3E50`
   → Remplacé par la classe `.page-title` (Slate 900, font-weight 900)

2. Boutons "Réinitialiser/Annuler" gris : `#BDC3C7` avec texte blanc
   → Remplacé par la classe `.btn-outline` (transparent, bordure grise)

3. Boutons "Supprimer" rouge plat : `#E74C3C` plein
   → Remplacé par la classe `.btn-danger` (fond rouge pâle, texte rouge vif)

4. Boutons "+" emoji (`➕`) avec style inline vert `#18BC9C`
   → Remplacé par la classe `.btn-quick-add` (Vert émeraude pastel)

5. CheckBox avec emoji `🖨` codé en dur
   → Emoji supprimé, texte propre "Imprimer Bon de Commande"

6. Tableaux sans `CONSTRAINED_RESIZE_POLICY`
   → Ajout systématique sur produits (x3), fournisseurs (x1)

DÉCISIONS ARCHITECTURALES - CSS (styles.css)
--------------------------------------------
Ajout de 4 classes globales en fin de fichier :

1. `.page-title` : Police 22px, poids 900, couleur #0F172A (Slate 900)
   Usage : Tous les titres de modules de l'application.

2. `.btn-danger` : Fond #FEF2F2 (Red 50), bordure #FECACA (Red 200),
   texte #DC2626 (Red 600). Hover assombrit légèrement.
   Usage : Tous les boutons de suppression destructrice.

3. `.btn-outline` : Fond transparent, bordure #CBD5E1 (Slate 300),
   texte #64748B (Slate 500). Hover : fond très pâle.
   Usage : Tous les boutons secondaires non-critiques.

4. `.btn-quick-add` : Fond #ECFDF5 (Emerald 50), bordure #6EE7B7,
   texte #059669 (Emerald 600). Taille de police 16px pour le "+".
   Usage : Boutons d'ajout rapide inline dans les formulaires.
